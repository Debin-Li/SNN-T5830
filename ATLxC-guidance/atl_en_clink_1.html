CLINK Utility

The CLINK utility consists of interfaces that are used to call the C program functions from the ATL program.

To call the C program functions, follow the procedures described below.

  1. Create an object file that includes a function to be called, by using the C program.

  2. Load the object file by using CALL CLOAD.

  3. Call the function by using EXEC CCALL.
