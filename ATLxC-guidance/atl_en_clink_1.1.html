CALL CLOAD

  1. Function

    This utility loads the object file of the C program to the memory.

  2. Syntax 

    

    CALL CLOAD(OBJNAME,ECODE,MODE)

    • OBJNAME

      Specify the name of the object file to be loaded.

      Specify the file name by using the STRING variable or a character constant.

      This argument cannot be omitted.

      For more information on how to create the object file, refer to How to Create the Object File.

    • ECODE

      Specify the variable that stores an error code when an error occurs.

      If the program ends correctly, -1 is stored.

      Specify the variable by using an integer variable.

      This argument can be omitted.

    • MODE

      Specify the mode that unloads the object file.

      Specify the mode by using an integer variable or integer constant.

      This argument can be omitted. The default value is 0.

      MODE Setting Range
      Setting range
      0 1

    CLOAD is used to unload the object file, which is specified in OBJNAME, to the memory.

    Only a file name can be specified in OBJNAME. No path can be specified.

    Object files are searched for in the following order:

    1. ADI_ADP1 environment variable

    2. The directory that is set by the Directory Linker

    3. DP0 environment variable

    If the specified object file is already loaded, the object file is not reloaded.

    By executing CLOAD more than once, multiple object files can be loaded.

    MODE Argument
    MODETiming of unload
    0 (Automatic mode)Unloads the object file when the program ends.
    1 (Manual mode)Does not unload the object file when the program ends.

    If any of the following conditions is satisfied, the object file is unloaded regardless of the mode.

    • When CUNLD is executed
