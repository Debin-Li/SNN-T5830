UTATL_ClinkReadVar

  1. Function

    This function reads ATL variables in the C program of the CLINK utility.

  2. Syntax 

    

    #include "UTATL.h"
    int UTATL_ClinkReadVar( void *atlvar , void  *dest   , int type , long *signword )

    • atlvar

      Specify the ATL variable address that was passed as an argument.

    • dest

      Specify the pointer that stores data of the ATL variable to be read.

      Specify the variable address that corresponds to the ATL variable data.

      Specify the address by casting it as void.

      The area is provided by the function that uses the variable.

      ATL variableRequired variable
      INTEGER[1]long xxx;
      INTEGER[2]long xxx;
      INTEGER[3]long xxx;
      INTEGER[4]long xxx[2];
      LOGICAL[1]unsigned long xxx;
      LOGICAL[2]unsigned long xxx;
      LOGICAL[3]unsigned long xxx;
      LOGICAL[4]unsigned long xxx[2];
      REAL[2]double xxx;
      REAL[3]double xxx;
      REAL[4]double xxx;
      CHARACTER[1]char xxx[2]; (*1)
      CHARACTER[2]char xxx[4]; (*1)
      CHARACTER[3]char xxx[6]; (*1)
      CHARACTER[4]char xxx[8]; (*1)
      STRING(n)char xxx[n+1]; (*2)
      GENERAL[3]long xxx; (*3)
      unsigned long xxx; (*4)
      double xxx; (*5)
      char xxx[6]; (*1)(*6)
      GENERAL[5]long xxx[2]; (*3)
      unsigned long xxx[2]; (*4)
      double xxx; (*5)
      char xxx[10]; (*1)(*6)

      • (*1) This will not be terminated with '\0'.

      • (*2) This will be terminated with '\0'.

      • (*3) Specify this when value of INTEGER type is stored in GENERAL variable.

      • (*4) Specify this when value of LOGICAL type is stored in GENERAL variable.

      • (*5) Specify this when value of REAL type is stored in GENERAL variable.

      • (*6) Specify this when value of CHARACTER type is stored in GENERAL variable.

    • type

      Variable type and size. Specify the following macro. 

      


    UTATL_VarTypeInte1  /* INTEGER[1] */
    UTATL_VarTypeInte2  /* INTEGER[2] */
    UTATL_VarTypeInte3  /* INTEGER[3] */
    UTATL_VarTypeInte4  /* INTEGER[4] */

    UTATL_VarTypeLogi1  /* LOGICAL[1] */
    UTATL_VarTypeLogi2  /* LOGICAL[2] */
    UTATL_VarTypeLogi3  /* LOGICAL[3] */
    UTATL_VarTypeLogi4  /* LOGICAL[4] */

    UTATL_VarTypeReal2  /* REAL[2] */
    UTATL_VarTypeReal3  /* REAL[3] */
    UTATL_VarTypeReal4  /* REAL[4] */

    UTATL_VarTypeChar1  /* CHARACTER[1] */
    UTATL_VarTypeChar2  /* CHARACTER[2] */
    UTATL_VarTypeChar3  /* CHARACTER[3] */
    UTATL_VarTypeChar4  /* CHARACTER[4] */

    UTATL_VarTypeString /* STRING */

    UTATL_VarTypeGene3  /* GENERAL[3] */
    UTATL_VarTypeGene5  /* GENERAL[5] */

      CAUTION

      When specified a model of a variable and size by mistake, it may cause memory destruction.

    • signword

      Specify the area in which the variable information is stored. Variable information is one top word of a variable area which the word size of the variable is three or five words and STRING variable. For more information, refer to Variable Type Declaration and Data Formats.

      Specify the address of the long-type area.

      The area is provided by the function that uses the variable.

      This argument is enabled when the word size of the variable is three or five words and STRING variable.

      If NULL is specified in this argument, no variable information is returned regardless of whether the variable information exists. (No error occurs.)

      Therefore, specify NULL if either of the following conditions is satisfied:

      • Except the word size of the variable is three and five words or STRING variable.

      • No variable information is required even if the word size of the variable is three or five words or STRING variable.

    • Return value

      Return valueDescription
      UTATL_CLINK_OKSuccess
      UTATL_CLINK_ERR_ADDRESSReturned when NULL is specified in the atlvar and dest arguments.
      UTATL_CLINK_ERR_TYPEReturned when the value specified in the type argument is out of the specified range.
